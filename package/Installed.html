Convert from a source file to the srcML format:

srcml main.cpp -o main.cpp.xml

Convert from the srcML format back to source code

srcml main.cpp.xml -o main.cpp.xml

Convert text code fragment

srcml --text="int a = b;" -l C++ -o fragment.cpp.xml

Note the programming language (C, C++, C#, or Java) must be specified for the –text option

Convert entire project to srcML

srcml --verbose project -o project.xml
